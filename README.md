# NativePDF

**NativePDF is a Windows desktop application for working with PDF documents.**

This repository is the official location for NativePDF downloads, release notes, documentation, and support information.

## Download

NativePDF 1.0.0 is currently undergoing final release validation.

The Windows installer will be available from the **Releases** section after testing and approval are complete.

> Do not download NativePDF installers from unofficial websites or third-party sources.

## System requirements

* Windows 11
* Windows 10 version 22H2 is supported on a compatibility basis
* 64-bit Windows computer

## Installation

Once NativePDF 1.0.0 is published:

1. Open the latest entry under **Releases**.
2. Download the NativePDF Windows setup `.exe`.
3. Verify that the version and publisher information are correct.
4. Run the installer.
5. Follow the installation instructions.

## Security

Official NativePDF releases will include:

* A Windows installer
* Version information
* SHA-256 checksum
* Release notes
* Publisher and signature information

The public installer will be published only after completing the release checks and Windows testing.

## Privacy

NativePDF is designed as a desktop application for processing PDF documents locally.

The complete privacy policy will be added before the first public release.

Support instructions and known limitations will be published with NativePDF 1.0.0.
